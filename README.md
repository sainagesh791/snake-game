# 🐍 Snake Game

A simple and responsive Snake Game built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

https://snake-game791.netlify.app/

## 📌 Features

- Classic snake gameplay
- Score tracking
- High score saved using Local Storage
- Responsive design
- Smooth animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure

```bash
snake-game/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

---

## ▶️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/snake-game.git
```

2. Open the project folder

3. Run `index.html` in browser
